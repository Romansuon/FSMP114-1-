# ФСМП 114 група: Протидія домашньому насильству

## Опис

Веб-проєкт студентів ФСМП 114 групи, присвячений темі протидії домашньому насильству. Сайт містить статистику, інформаційні матеріали, інтерактивні графіки, презентації та команду учасників.

## Особливості

- Інтерактивна статистика з перемиканням типів діаграм (стовпчикова, кругова, лінійна)
- Адаптивний дизайн для мобільних пристроїв
- Персональні модальні вікна з інформацією про учасників проєкту
- Підтримка різних кольорових тем та градієнтів
- Анімований ефект світіння для основних блоків

## Технології

- HTML5, CSS3 (Bootstrap 5, кастомні стилі)
- JavaScript (Chart.js для графіків)
- Font Awesome, Google Fonts

## Як запустити

1. Скачайте або клонуте репозиторій:
    ```
    git clone https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git
    ```
2. Відкрийте файл `index.html` у браузері.

## Структура проекту

- `index.html` — головна сторінка з усіма секціями
- `/assets/` — зображення, іконки, додаткові файли (якщо є)
- `README.md` — цей файл
